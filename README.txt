###

This directory contains phased nDNA sequence data, mtDNA data, beast input files, morphological data, and a structure input file


~/Dornburgetal_2016_MolEcolData/Beast_files/ contains the beast files for the bGMYC analyes

~/Dornburgetal_2016_MolEcolData/Morpho_Data/ contains a .csv of morphological data measurements

~/Dornburgetal_2016_MolEcolData/mtDNA/ contains the ND2 sequence data

~/Dornburgetal_2016_MolEcolData/Phased_nDNA contains the phased nDNA files in .fasta and .phy format

~/Dornburgetal_2016_MolEcolData/Structure_Input/ contains an input file of all data for running structure
